项目二：Spring Boot + Vue 用户信息管理应用（JSON 存储）
技术栈仅使用 Spring Boot + Vue，用 JSON 文件存储用户数据，不添加数据库或第三方依赖。
后端：在 src/main/resources 下创建 users.json，字段为 id,name,age,email；开发用户列表查询、新增、编辑、删除接口，端口 10021；接口直接读写 JSON 文件，确保数据持久化。
前端：用 Vue 开发用户列表页，包含添加用户弹窗和编辑功能；列表展示所有用户信息，支持点击删除按钮删除用户；通过 Axios 与后端接口交互，操作后实时刷新列表。
本地运行：后端启动 Spring Boot 项目，前端启动服务后访问localhost:10022，无需配置数据库，直接通过 JSON 文件存储用户数据。
